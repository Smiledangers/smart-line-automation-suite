# 智慧型 LINE Bot + 後台 Dashboard + 自動化爬蟲管線 Demo 專案
# 完整專案結組與說明

本專案是一個使用現代 Python 技術棧建置的生產級全棧應用程式，包含：
- LINE Bot 互動
- 後台管理 Dashboard
- 自動化爬蟲管線
- AI 代理整合 (LangGraph)
- Docker Compose & Kubernetes 部署
- GitHub Actions CI/CD
- 完整測試框架

## 快速開始
### 先決條件
- Docker 與 Docker Compose
- Python 3.11+ (選擇用於本地開發 без Docker)
- Node.js 16+ (如果使用前端框架，但本專案目前僅提供後端 API)

### 使用 Docker Compose（推薦用於開發）
```bash
# 複製環境變數範例
cp .env.example .env
# 根據需要編輯 .env（特別是 LINE 和 AI 的金鑰）

# 建置並啟動所有服務
docker-compose up -d

# 執行資料庫遷移
docker-compose exec web alembic upgrade head

# 建立第一個管理員使用者
docker-compose exec web python scripts/create_superuser.py

# 存取應用程式
# API 文件: http://localhost:8000/docs
# 替代文件: http://localhost:8000/redoc
# 健康檢查: http://localhost:8000/health
```

### 使用 Kubernetes
```bash
# 應用所有 manifests
kubectl apply -f deploy/k8s/

# 或使用 Helm (如果已準備好圖表)
# helm install demo-bot ./deploy/helm/demo-bot
```

### 本地開發（無 Docker）
```bash
# 建立虛擬環境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安裝依賴
pip install -r requirements/base.txt
pip install -r requirements/dev.txt  # 包含測試和開發工具

# 複製環境變數
cp .env.example .env
# 編輯 .env

# 初始化資料庫
python scripts/init_db.py

# 執行遷移
alembic upgrade head

# 建立管理員使用者
python scripts/create_superuser.py

# 启动应用
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## 部署選項
本專案提供多種部署方式：

1. **Docker Compose** - 適合開發、測試和小規模生產環境
2. **Kubernetes** - 適合需要擴縮和高可用性的生產環境
3. **Render.com** - 使用 deploy/render.yaml 透過 Render.com 部署
4. **Railway.app** - 使用 deploy/railway.json 透過 Railway.app 部署
5. **傳統 VPS** - 使用 Docker 或直接在伺服器上執行

## 環境變數說明
請參閱 `.env.example` 檔案以獲取所有必要的環境變數說明。重要的變數包括：

- `DATABASE_URL`: PostgreSQL 連線字串
- `REDIS_URL`: Redis 連線字串
- `LINE_CHANNEL_ACCESS_TOKEN`: LINE Bot 通道存取權杖
- `LINE_CHANNEL_SECRET`: LINE Bot 通道密鑰
- `OPENAI_API_KEY`: OpenAI API 金鑰（用於 LangGraph）
- `SECRET_KEY`: 用於 JWT 簽名的密鑰
- `FIRST_SUPERUSER`: 首個管理員使用者的 Email
- `FIRST_SUPERUSER_PASSWORD`: 首個管理員使用者的密碼

## 注意事項
1. 在生產環境中，請務必變更 `.env.example` 中的預設值，特別是 `SECRET_KEY`、`LINE_CHANNEL_ACCESS_TOKEN`、`LINE_CHANNEL_SECRET` 和 `OPENAI_API_KEY`。
2. 對於 LINE Bot，您需要在 LINE Developers Console 中設定 Webhook URL 為 `https://your-domain.com/api/v1/line/webhook`。
3. 對於 AI 功能，如果您沒有 OpenAI API 金鑰，您可以設定使用本地 LLM 模型（需修改 ai_service.py 和相關設定）。
